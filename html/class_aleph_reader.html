Detailed Description

A class to read ALEPH ROOT data files converted with paw2root.py.

This class provides an accessor to load the ROOT files into. It can be used in two ways: 1) AlephReader controls the selection, histogramming and event loop. In this instance, use SetSelection(), AddVariable(), Process() and Write() 2) The user controls the everything and only uses this class to access the raw variables In this instance, use GetEvent() and the individual variable accessors.

This class can be used in interpretive ROOT root - .L AlephReader.cxx++ root - AlephReader reader root - reader.SetFilename(...) root - reader.SetTreename(...) root - reader.SetSelection(...) etc.. or it can be compiled against standalone C++ (or potentially used inside another macro).

void AlephReader::Process ( )

Process()

This function loops over all the events in the tree internally and applies a selection (defined by SetSelection()). The histograms which are defined with AddVariable() will be filled if the event passes the given selection. The output can be saved with Write().
